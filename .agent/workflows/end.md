# 🏁 Workflow: End Session (/end)

Este flujo asegura que todo el progreso y conocimiento generado se guarde correctamente.

## Pasos

1. **Extraer Aprendizajes**: Identificar nuevas decisiones, patrones o soluciones descubiertas durante la sesión.
2. **Actualizar Memoria**: Registrar estos aprendizajes en `.context/MEMORY.md`.
3. **Actualizar Docs**: Ejecutar el comando `/actualizar` para poner al día `ARCHITECTURE_SUMMARY.md` y `README.md`.
4. **Resumen Final**: Proporcionar un resumen de los cambios realizados y los próximos pasos pendientes.
5. **Cruce de Datos**: Verificar que no queden archivos temporales o comandos de debug activos.
