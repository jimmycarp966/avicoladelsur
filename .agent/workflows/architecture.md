# 🏗️ Workflow: Entender Arquitectura (/architecture)

Este flujo proporciona una inmersión profunda en la estructura técnica del proyecto.

## Pasos

1. **Lectura Fractal**:
    - Nivel 1: `ARCHITECTURE_SUMMARY.md` (Vista general).
    - Nivel 2: `.framework/LAWS.md` (Reglas).
    - Nivel 3: `ARCHITECTURE.md` (Detalles técnicos y esquemas DB).
2. **Exploración de Directorios**: Listar `src/app`, `src/actions` y `supabase/migrations`.
3. **Mapeo de Dependencias**: Identificar integraciones críticas (Google Cloud, Twilio, Supabase).
4. **Resumen de Módulos**: Identificar los 5 dominios principales (Ventas, Reparto, Almacén, Tesorería, Sucursales).
